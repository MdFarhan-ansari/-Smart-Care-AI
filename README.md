🏥 Smart Care AI

Smart Care AI is an AI-powered healthcare management system built using Python, Streamlit, MongoDB, and Machine Learning. The platform helps users monitor their health, analyze symptoms, predict diseases, calculate BMI, manage medical records, and receive AI-powered health assistance.

---

🚀 Features

👤 User Management

- User Registration
- Secure Login System
- Session Management
- Personal Health Dashboard

🩺 Symptom Checker

- Enter symptoms
- AI-based symptom analysis
- Possible disease suggestions

🤖 Disease Prediction

- Machine Learning-powered prediction
- Health risk assessment
- Personalized recommendations

📏 BMI Calculator

- Automatic BMI calculation
- Health category detection
- Fitness insights

📂 Medical Report Management

- Upload medical reports
- Store and manage health records
- Easy access to reports

💬 AI Health Assistant

- Healthcare chatbot
- Basic medical guidance
- Health-related Q&A

📊 Health Analytics

- User health overview
- Health statistics
- Personalized dashboard insights

⚙️ Installation

Clone Repository

cd SmartCareAI

Install Dependencies

pip install -r requirements.txt

Run Application

streamlit run app.py
